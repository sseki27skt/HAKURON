# 博士論文公開リポジトリ

[![DOI](https://zenodo.org/badge/1080217829.svg)](https://doi.org/10.5281/zenodo.17576727)

このリポジトリでは、東京大学大学院人文社会系研究科に提出された博士論文『楽譜記述の機械可読化による無形文化財の保護・継承・研究 ―唐楽中小曲に含まれる三管の譜を対象としたデジタル版雅楽譜の構築を通して―』を公開しています。公開用Webサイトはhttps://sseki27skt.github.io/HAKURON/ です。公開用Webサイトに提出版のPDFへのリンクも掲載しています。

## Web版の注意

Web公開に際して一部のコンテンツを更新しています。Webページ版にはページ番号がありませんので、博士論文本文の引用が必要な場合は提出版のPDFもあわせてご参照ください。

### 主な更新点

- 序に対して章番号を付与したことで、全体の章番号が更新されています。
- 図2.1 をMeramid記法で書き換えています。図の構成に変更はありません。

## Webサイト構築方法
WebサイトはLuaLaTeXで作成した原稿を元にmarkdawnを作成し、[Quarto](https://quarto.org/)を使用してレンダリングを行っています。また、執筆時には[jecon.bst]([url](https://github.com/ShiroTakeda/jecon-bst))を使用させていただきました。Webページ版の参考文献についても、jecon.bstを使用してコンパイルした際に出力されるbblファイルを整形して流用しています。
